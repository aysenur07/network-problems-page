# Network Problems Project

This project includes an *HTML page explaining 10 current network-related issues and solutions. It also contains corresponding **PDF documents, **images, and **Cisco Packet Tracer simulation files* for each problem.

---

## Project Structure
├── images/ # Visuals for the problems and solutions
├── pdfs/ # PDF explanations
├── simulations/ # Cisco Packet Tracer (.pkt) files
├── styles.css # CSS for styling the page
├── problems.js # JavaScript data for the problems
├── mypage.html # Main HTML file (entry point)


## ⚠⚠⚠ Important Notice ⚠⚠⚠

To run the project correctly:

> *Extract the ZIP file before opening anything.*  
> If you try to open the HTML file or any other files directly from inside the ZIP archive, images, styles, and links *will not work properly*.

### ✅ Correct usage:

1. Extract the ZIP archive to a folder on your computer.
2. Open the mypage.html file in your browser.



## How to Use

- Use the *sidebar* to select a network problem.
- The main area will display the explanation.
- Click the *PDF* button to open the detailed document for that problem.
- Download the .pkt file from the provided link to view the simulation in *Cisco Packet Tracer*.



## Requirements

- A modern web browser (Chrome, Edge, Firefox, etc.)
- [Cisco Packet Tracer] (to open simulation files)
